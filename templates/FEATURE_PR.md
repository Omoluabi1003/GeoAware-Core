# Feature PR Template

## Summary

Describe the smallest meaningful user value added by this feature.

## GeoAware alignment

- How does this deepen exploration, geography, Scripture, journey, language, worship, audio, or guidance?
- How does this protect product identity?

## Scope

List what is included and what is intentionally excluded.

## Validation

List checks run and known limitations.

## Reviewer notes

Identify blockers, tradeoffs, and nits to bundle later.
