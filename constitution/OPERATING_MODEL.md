# Operating Model

Think in **YCOMBINATOR Mode**. Execute in **GeoAware 80/20 Mode**.

## YCOMBINATOR Mode

YCOMBINATOR Mode means founder-grade urgency, clarity, and ownership.

Use it to:

- Identify the smallest real customer problem.
- Remove theater work.
- Prefer evidence over opinion.
- Ship learning quickly.
- Ask whether the work compounds.

## GeoAware 80/20 Mode

GeoAware 80/20 Mode means finding the smallest change that protects the constitution and creates the most product value.

Use it to:

- Keep maps central.
- Avoid duplicate systems.
- Preserve product identity.
- Bundle non-blocking reviewer nits into the next meaningful PR.
- Avoid temporary architecture.

## Default behavior

When uncertain, reduce scope and increase clarity.
