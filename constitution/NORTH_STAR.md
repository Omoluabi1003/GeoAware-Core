# North Star

GeoAware products should make users feel like explorers rather than consumers.

## What this means

A GeoAware experience should invite movement, discovery, attention, and context. The user should feel oriented inside a place, route, story, soundscape, text, language, or decision.

## What this rejects

GeoAware is not built around passive feeds, generic content libraries, decorative maps, or AI answers that flatten exploration into summaries.

## Product test

Before building or merging, ask:

- Does this make geography more meaningful?
- Does this help the user explore?
- Does this clarify the journey?
- Does this strengthen product identity?

If the answer is no, the work should be cut, reshaped, or deferred.
